My Homelab

Current Hardware

  • 4 Dell Optiplex 7020 Towers
  • Cisco Catalyst 3750G 48 Port Layer 3 Switch
  • Cheap Linksys Router

Currently I have 2 of the desktops repurposed as ESXi hypvervisors with vCenter. The other desktop is running pfSense as my router/firewall and also as an OpenVPN server. I have the Cisco 3750G doing my inter-vlan routing and then have a routed port set up back to the pfSense box where I have static routes setup so that certain VLANs can reach the internet. I have another repurposed desktop not shown that I'm planning to use for iSCSI storage with vSphere, I'm just waiting to buy NICs.

The old linksys router has been put in wireless bridge mode to act as an access point until I get my hands on some better equipment

I have around 10 virtual machines that are running various things including:

  • Active Directory Domain Services
  • Active Directory Certificate Services
  • AD is also doing internal DNS resolution and DHCP, forwarders set to Google DNS
  • Docker Hosts
  • Windows Admin Center
  • Wiki Server running Wiki.js (to keep track of all this stuff!)
  • Network Monitor - planning to use InfluxDB, Graphana
  • Nextcloud File Server - still setting up
  • This website is currently hosted on Linode but I plan to move it to my homelab in the future
